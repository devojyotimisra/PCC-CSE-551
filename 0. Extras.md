# Connect to UIT Oracle Server

1. `Open 'Secure Shell Client' on Windows`
2. `Click the 'File' tab from the navigation bar.`
3. `Click the 'Quick Connect' option from the 'File' Tab`
4. `Fill 'Host Name' with '192.168.153.16'`
5. `Fill 'User Name' with 'uit'`
6. `Keep the rest of the options as it is`
7. `Click on the 'Connect' button`
8. `Fill 'Password' with 'uit'`
9. `Type 'ssh oracle@192.168.153.16' in the shell`
10. `Type 'oracle' as the password`
11. `Run the 'sqlplus' command`
12. `Use 'User Name' as 'scott'`
13. `Use 'Password' as 'tiger'`
14. `You are connected to the Oracle Server Successfully`
